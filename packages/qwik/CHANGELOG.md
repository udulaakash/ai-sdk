# @ai-sdk/qwik

## 1.0.0

### Major Changes

- Initial release of AI SDK Qwik integration
- Added `useChat` hook for chat interfaces
- Added `useCompletion` hook for text completion
- Added `experimental_useObject` hook for structured object streaming
- Added `Chat` class for managing chat state with Qwik signals
- Full API compatibility with React package but adapted for Qwik's reactive system
